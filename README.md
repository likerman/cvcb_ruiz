# cvcb_ruiz — Tesis de Licenciatura (UBA)

Repositorio de trabajo asociado al **Trabajo Final de Licenciatura**:

**“Evaluación del potencial geotermoeléctrico del Complejo Volcánico Cerro Blanco (Catamarca) mediante modelado numérico”**  
Departamento de Ciencias Geológicas, Facultad de Ciencias Exactas y Naturales, Universidad de Buenos Aires.  
Presentado en **febrero de 2026**.  
Autora: **Luciana Ruiz**. Dirección: **Dra. María Clara Lamberti**. Codirección: **Dr. Jeremías Likerman**.

## Resumen
El trabajo evalúa el potencial geotermoeléctrico del CVCB integrando:
- síntesis de antecedentes geológicos,
- **cartografía geológica 1:90.000**,
- **modelos térmicos numéricos 2D** para analizar la evolución térmica del sistema y estimar potencial geotermoeléctrico bajo supuestos explícitos.

## Estructura del repositorio
- `UW/`  
  Código/notebooks y recursos asociados al modelado numérico (Underworld / flujos de simulación).
- `plot/`  
  Scripts/notebooks para post-proceso y figuras.
- `resultados_potencial.csv`  
  Tabla de resultados procesados (potenciales/escenarios).
